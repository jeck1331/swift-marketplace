# Swift Marketplace
A high-load server ECOM project for marketplaces with a microservices architecture, Redis caching and Kafka.
## Stack
### Language and Frameworks
- C# .NET 10
- ASP.NET
### Database
- main: PostgreSQL
- cache: Redis
### Tools and Libraries
- FluentMigrator
- Npgsql
- Dapper
- Confluent.Kafka
- ...
### Message Broker
- Kafka
### CI\CD
- Docker
